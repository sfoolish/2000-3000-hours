# Mac Docker 使用

## 常用命令

启动命令

    $ boot2docker init
    $ boot2docker start

mac 本地测试

    $ export DOCKER_HOST=tcp://$(boot2docker ip 2>/dev/null):2375
    $ docker run ubuntu echo hello world

关闭命令

    $ boot2docker stop
     
登入虚拟机

    $ boot2docker ssh

帮助命令

    $ boot2docker -h
