## 死循环

```sh
    
    while true
    do
        ping -c 1 www.baidu.com > /dev/null 2>&1
        if [ $? -ne 0 ]; then
            echo "network is not reachable" $?
            ifconfig en1 down
            sleep 3
            ifconfig en1 up
        fi
    
        sleep 100
    done

```
