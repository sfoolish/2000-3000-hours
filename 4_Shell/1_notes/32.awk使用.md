## awk 字符串截断

```sh
echo "/aaa/bbb/ccc" | awk -F '/' '{print $NF}'
```

> ccc

```sh
echo hello:world:sf | awk -F":" '{ print $1 $3}' 
```

> hellosf

```sh
echo hello:world:sf | awk -F":" '{ print $1" " $3}' 
```

> hello sf

```sh
echo hello:world:sf | awk -F":" '{ print "username: " $1 "\t\tuid:" $3 }'
```

> username: hello		uid:sf

fi
