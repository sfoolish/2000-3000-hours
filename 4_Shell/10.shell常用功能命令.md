## shell 功能命令

### 代码行数查看

```sh

    $ find ./ -name "*.c"  | xargs wc -l | sort -n

```

搜索 *.c, *.h, *.cc 三类文件

``` sh

    $ find . \( -name "*.h" -o -name "*.c" -o -name "*.cc" \) -print | xargs wc -l 

```

## ps 打印指定进程的线程信息

```sh

    ps -mp pid -o THREAD,tid,time

```
