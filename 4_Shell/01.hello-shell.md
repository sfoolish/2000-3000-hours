## shell 命令制作

    ; 创建 hello 命令文件
    $ vim cmd_hello
    
    ; 简析 hello 命令
    $ cat cmd_hello
    ```
        #!/usr/bin/env bash
    
        echo 'hello shell'
    ```
    `#!/usr/bin/env bash` 表示通过 env 寻找 bash 来解释执行后面的脚本。
    `echo 'hello'` 通过标准输出字符串 hello 。    
    
    ; 添加执行权限
    $ chmod +x cmd_hello
    
    ; 运行命令
    $ ./cmd_hello
    ```
        hello shell
    ```
