# Go IDE

## Sublime Text

### Sublme Text 插件安装

	command + shift + p 
		-> 输入 pcip
			-> 输入 GoSublime / SidebarEnhancements / Go Build

### GoSublime 常用功能

打开命令行窗口

	command + 9

关闭命令行窗口

	ESC

go fmt 代码整理

	command + s 保存文件的时候默认会执行 go format

命令行中执行 shell 示例

	sh echo 'hello'

帮助文档查看

	help
